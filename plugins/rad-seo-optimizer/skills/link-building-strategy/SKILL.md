# Link Building Strategy Skill

Build a complete link acquisition plan from zero experience to earning high-quality backlinks. This skill walks through seven phases and outputs a 12-week action plan.

---

## Phase 1: Current Link Profile Assessment

Start by understanding where the site stands today.

### Step 1 — Gather Backlink Data

If the user provides exported data from Ahrefs, Moz, or Semrush, parse it and summarize. Otherwise, ask the user to:

1. Use a free tool (Ahrefs Webmaster Tools, Moz Link Explorer free, or Ubersuggest free tier).
2. Export their backlink list or provide these numbers:
   - Total referring domains
   - Total backlinks
   - Domain authority / domain rating
   - Top 10 linking domains by authority

### Step 2 — Quality Distribution

Classify every known linking domain into tiers:

| Tier | Domain Authority | Quality Label | Typical Examples |
|------|-----------------|---------------|------------------|
| A | 60+ | Premium | Major publications, .edu, .gov |
| B | 30-59 | Solid | Industry blogs, established sites |
| C | 10-29 | Low-mid | Small blogs, directories |
| D | 0-9 | Low/Risky | Spam farms, PBNs, foreign junk |

Report the percentage split across tiers.

### Step 3 — Toxic Link Check

Flag links that match any of these patterns:
- Domains with DA < 5 and no real content
- Sites in unrelated foreign languages with no topical connection
- Known link farms or PBN patterns (many outbound links, thin content)
- Exact-match anchor text appearing suspiciously often

Recommend submitting a Google Disavow file if Tier D links exceed 15% of total.

### Step 4 — Benchmark Score

Produce a simple scorecard:

```
Domain Authority:        [score] / 100
Referring Domains:       [count]
Tier A links:            [count] ([%])
Tier B links:            [count] ([%])
Tier C links:            [count] ([%])
Tier D (toxic) links:    [count] ([%])
Link Velocity (monthly): [estimate]
Overall Health:          [Good / Needs Work / Critical]
```

## Phase 2: Link Gap Analysis

### Step 1 — Identify Competitors

Use the 3 competitor URLs provided, or suggest the top 3 ranking sites for the user's primary keyword.

### Step 2 — Compare Profiles

Build a comparison table:

| Metric | Your Site | Competitor 1 | Competitor 2 | Competitor 3 |
|--------|-----------|-------------|-------------|-------------|
| Domain Authority | | | | |
| Referring Domains | | | | |
| Tier A Links | | | | |
| Link Velocity | | | | |

### Step 3 — Find the Gap Sites

Identify domains that link to at least 2 competitors but not to the target site. These are "warm" prospects because they already link to similar content.

Sort by:
1. Domain authority (higher = better)
2. Topical relevance (same industry = better)
3. Link type (editorial mention > directory > comment)

### Step 4 — Score and Prioritize

Rate each opportunity on a 1-5 scale:

- **Relevance** — How closely does the linking site match the target niche?
- **Authority** — What is the site's DA?
- **Accessibility** — Does the site accept contributions, have a contact form, or list an editor?
- **Link type potential** — Could a contextual, in-content link be earned?

Priority = (Relevance + Authority + Accessibility + Link Type) / 4. Pursue 4.0+ first.

## Phase 3: Linkable Asset Audit

Links cannot be earned without something worth linking to.

### Existing Content Review

Evaluate the user's current content for link-worthiness. Strong linkable assets include:

- **Original research or data** — Surveys, studies, industry benchmarks
- **Comprehensive guides** — "Ultimate guide to X" that genuinely earns the title
- **Free tools or calculators** — Interactive resources people reference
- **Templates and frameworks** — Downloadable resources others cite
- **Infographics or visual data** — Shareable visual content
- **Controversial or unique opinions** — Strong takes that spark discussion

### Content Gap: What to Create

Recommend 3-5 new "citation magnet" ideas based on the user's industry:

1. **Original Data Play** — Survey your customers or analyze your internal data. Publish findings with charts. Journalists and bloggers cite original data.
2. **The Definitive Resource** — Create the single best page on a subtopic. Make it 3x better than what currently ranks.
3. **Free Tool or Calculator** — Even a simple spreadsheet template hosted as a Google Sheet earns links.
4. **Named Framework** — Create a branded methodology (e.g., "The 3R Content Framework"). Named concepts get cited and linked.
5. **Annual Report or Index** — Publish a yearly state-of-the-industry report. It becomes a recurring link magnet.

### Branded Strategy Opportunities

Help the user coin a term or framework unique to their brand. Requirements:
- Simple and memorable (3-5 words max)
- Genuinely useful as a mental model
- Easy to reference in other people's content
- Searchable (someone who hears it can Google it and find you)

## Phase 5: LLM Co-Citation Strategy (AEO-Specific)

Modern SEO must account for AI-generated answers. Links are not just for Google rankings anymore; they train the data that AI models use to recommend brands.

### Goal: Get Mentioned Alongside Industry Leaders

1. **Publish comparison content** — Create "X vs Y vs [Brand]" articles. AI models ingest these and learn to associate the brand with established players.
2. **Reddit and Quora presence** — Answer questions in relevant niche threads. Mention the tool/service naturally where relevant. These platforms are heavily used as AI training data.
3. **Earn roundup mentions** — Get included in "Top 10 tools for X" and "Best resources for Y" articles. AI models treat these lists as authority signals.
4. **Third-party reviews** — Encourage users to leave reviews on G2, Capterra, Trustpilot, or niche review sites. AI models reference these aggregators.
5. **Build cross-platform consensus** — If 5+ independent sources mention the brand in a similar context, AI models are more likely to recommend it. Aim for consistent messaging across all earned mentions.

## Phase 6: The 50/50 Plan

Link building fails when only creating content or only doing outreach. Split effort evenly.

### 50% — Creating Link-Worthy Content

- Publish 1 linkable asset every 2-4 weeks
- Each piece should target a specific tactic (e.g., a data study for Digital PR, a guide for Skyscraper)
- Quality over quantity: one excellent piece beats ten mediocre ones

### 50% — Active Promotion and Outreach

- Send outreach emails every week without exception
- Track every email in a simple spreadsheet (date, recipient, tactic, status, result)
- Follow up exactly once, 5-7 days after the first email

### Weekly Targets by Available Hours

| Monthly Hours | Weekly Outreach Emails | Monthly Content Pieces | Expected Links/Month |
|--------------|----------------------|----------------------|---------------------|
| 10 | 10-15 | 1 | 1-3 |
| 20 | 20-30 | 1-2 | 3-6 |
| 40 | 40-60 | 2-3 | 6-12 |
| 60+ | 60-100 | 3-4 | 10-20 |

## Output: 12-Week Link Building Plan

Generate a week-by-week plan tailored to the user's available hours and selected tactics.

### Weeks 1-2: Foundation
- Complete link profile assessment (Phase 1)
- Run competitor link gap analysis (Phase 2)
- Audit existing content for linkable assets (Phase 3)
- Set up outreach tracking spreadsheet (columns: Date, Prospect, URL, DA, Tactic, Email Sent, Follow-up, Result)
- Create target list of 50 prospects from gap analysis
- **Outreach target:** 0 emails (preparation phase)
- **Content target:** Outline 1 linkable asset

### Weeks 3-4: First Outreach
- Begin with highest-success-rate tactics: link reclamation + testimonials
- Send first batch of outreach emails
- Publish first linkable asset
- Submit to 3-5 link roundups
- Sign up for HARO or similar journalist request service
- **Outreach target:** 15-25 emails per week
- **Milestone:** 2-4 links earned

### Weeks 5-6: Scale Up
- Launch first skyscraper or broken link campaign
- Follow up on all unanswered emails from weeks 3-4
- Respond to 5+ HARO queries per week
- Start engaging in Reddit/Quora threads (Phase 5)
- **Outreach target:** 20-30 emails per week
- **Content target:** Publish second linkable asset
- **Milestone:** 5-10 total links earned

### Weeks 7-8: Diversify
- Add resource page outreach to the active tactics
- Send first guest post pitches (3-5 pitches)
- Write and submit 2-3 testimonials for tools the business uses
- Analyze what is working — double down on best-converting tactic
- **Outreach target:** 25-35 emails per week
- **Milestone:** 10-18 total links earned

### Weeks 9-10: Optimize
- Review outreach response rates — refine subject lines and templates
- Publish third linkable asset (consider original data or a free tool)
- Pitch 2-3 podcasts in the niche
- Create comparison content for LLM co-citation (Phase 5)
- **Outreach target:** 25-35 emails per week
- **Content target:** 1 new asset + 1 content transformation
- **Milestone:** 18-28 total links earned

### Weeks 11-12: Systematize
- Document the repeatable process for each tactic
- Build a recurring monthly calendar of outreach activities
- Set up Google Alerts for brand mentions (link reclamation pipeline)
- Plan next quarter's linkable assets based on what earned the most links
- **Outreach target:** 30-40 emails per week
- **Milestone:** 25-40 total links earned

### 12-Week Summary Targets

| Metric | Conservative | Moderate | Aggressive |
|--------|-------------|----------|-----------|
| Total outreach emails | 150 | 300 | 500+ |
| Links earned | 10-15 | 20-30 | 35-50 |
| DA improvement | +2-5 | +5-10 | +8-15 |
| Linkable assets created | 2-3 | 3-4 | 5-6 |
| Referring domains added | 8-12 | 15-25 | 30-45 |

## Key Reminders

1. **Personalize every email.** Mention something specific about their site. Generic mass emails get ignored and damage reputation.
2. **Track everything.** Progress cannot be measured without data. Log every outreach email and its outcome.
3. **Follow up once.** A single follow-up 5-7 days later is acceptable. More than that is spam.
4. **Focus on relevance over authority.** A DA 30 link from a perfectly relevant site often outperforms a DA 60 link from an unrelated one.
5. **Be patient.** Link building compounds. Months 1-2 feel slow. Months 3-6 is where momentum builds.
6. **Never buy links.** Purchased links violate Google's guidelines and create long-term risk. Earn them.
