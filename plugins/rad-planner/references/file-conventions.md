# File Conventions

The canonical specification of the RAD 8-doc standard — covering PRD.md, ARCHITECTURE.md, ASSUMPTIONS.md, DECISIONS.md, PLAN.md, CLAUDE.md, HANDOFF.md, session-log.md, plus `tasks.md` and `CLAUDE-FRAGMENT.md` — lives at the repository root:

→ **[`docs/file-conventions.md`](../../../docs/file-conventions.md)** (canonical source)

Both rad-planner and rad-session reference the same convention. If you're editing the conventions, edit the canonical file — don't fork the copy here.
