# Workflows & Checklists — Complete Reference

## The 30-Day Beginner Plan

For new users just getting started with a digital note-taking app. Commit to your chosen platform for 30 days and focus on getting just 10% more value from your existing information habits.

### Step 1: Identify Your Most Important Inputs
- Create a note titled "My most important inputs"
- List primary information sources (emails, podcasts, meetings, articles, books, social media)
- Rate each by value (high/medium/low) and volume
- This helps consciously prioritize note-taking efforts

### Step 2: Capture What Resonates
- Create a note titled "Quick Capture"
- When you encounter interesting information, don't act — just write it down
- Builds your "noticing muscles"
- Trains you to avoid reacting urgently to every input

### Step 3: Reframe Your Inputs
- Create six foundational notes/folders: Tasks, Read/Watch Later, Projects, Areas, Resources, Archives
- Sort Quick Capture items into these categories
- Separates actionable to-dos from reference material

### Step 4: Begin a Daily Practice
- **Daily:** Save resonant information to Quick Capture; review task list
- **2-3x per week:** Sort Quick Capture into the six categories
- **Weekly:** Review Projects and Areas; add new tasks or archive completed projects

## Quick-Start: 10-Minute PARA Setup

For users who want to start immediately. This is the minimum viable Second Brain.

### Minute 1-3: Archive Everything

Move ALL existing files, notes, and documents into a single folder:
- Name it: `Archive [today's date]` (e.g., "Archive 2026-03-22")
- Include: Desktop files, Downloads folder, Documents, notes inbox, cloud drive root
- Do NOT sort anything. Just move it all into the archive folder.
- This instantly gives you a clean workspace.

### Minute 3-5: Create the PARA Structure

Create exactly 4 top-level folders:
```
📁 1-Projects
📁 2-Areas
📁 3-Resources
📁 4-Archives
```

The numbers keep them sorted correctly in any file manager. Mirror these exact 4
folders in every tool you use:
- Notes app (Notion, Obsidian, Apple Notes)
- File system (Documents folder)
- Cloud drive (Google Drive, OneDrive, Dropbox)
- Optionally: Email folders, bookmarks

### Minute 5-8: List Active Projects

Write down everything you're actively working on that has:
- A specific goal or outcome (what does "done" look like?)
- A deadline or timeframe (when does it need to be done?)

Create a subfolder in `1-Projects` for each one.

### Minute 8-10: Quick Sanity Check

- Do you have between 10 and 15 projects? If way more, some are probably "false
  projects" (dreams without deadlines). Move them to Areas or Resources.
- Does each project have a clear outcome? "Get healthy" is an Area, not a Project.
  "Complete Couch-to-5K program by June" is a Project.

### You're Done

Your Second Brain is live. From now on:
- When you encounter something useful, save it in the relevant project folder
- When you start a new project, create a folder for it
- When you finish a project, move its folder to Archives
- Do a weekly review every 3-7 days to keep things current

Graduate to the 30-Day Beginner Plan above for a more structured ramp-up.

## Project Kickoff Checklist

Run this whenever you start a new endeavor to recycle past knowledge before starting from scratch.

1. **Capture current thinking**
   - Create a blank note in a new project folder
   - Brainstorm: What do I already know? What do I need to find out? What is my goal? Who can I talk to? What can I read?

2. **Review existing PARA folders**
   - Scan your current folders for templates, outtakes, or related notes
   - Check Areas and Resources for material that could apply

3. **Search for related terms**
   - Use your note app's search for specific keywords
   - Unearth ideas buried in unexpected places

4. **Move or tag relevant notes**
   - Relocate found notes into the new project folder
   - Or tag them for easy access during this project

5. **Create an outline (Archipelago of Ideas)**
   - Pull gathered material into a logical progression
   - Sequence your stepping-stones to form a plan

### Bonus Kickoff Actions
- Answer premortem questions: "What is most likely to fail?" "What do I not know?"
- Define success criteria
- Communicate goals to stakeholders
- Schedule a kickoff meeting

## Project Completion Checklist

Run this to harvest reusable knowledge before archiving.

1. **Mark the project complete**
   - Check off final tasks in your task manager

2. **Cross out the goal**
   - Move the goal to a "Completed" section
   - Maintain an inspiring inventory of victories

3. **Review and move Intermediate Packets**
   - Identify reusable documents, templates, agendas, or research
   - Move them to relevant Project, Area, or Resource folders

4. **Archive the project folder**
   - Move to Archives across all platforms (notes app, file system, cloud)
   - Keeps active workspace uncluttered

5. **Add a status note (if paused/canceled, not completed)**
   - Write where you left off, why it was paused, and lessons learned
   - This "Hemingway Bridge" makes it easy to resume months or years later

### Bonus Completion Actions
- Answer postmortem questions: "What did I learn? What could be improved? What went well?"
- Communicate the outcome to stakeholders
- Evaluate return on investment
- Celebrate the accomplishment

## Weekly Review Checklist

Perform every 3-7 days to clear digital workspaces and maintain momentum.

1. **Clear email inbox**
   - Process lingering emails
   - Action items go to task manager
   - Reference material goes to notes app

2. **Check calendar**
   - Review past week for follow-ups
   - Review upcoming week for preparation needed

3. **Clear computer desktop**
   - Move accumulated files into appropriate PARA folders

4. **Clear notes inbox**
   - Give newly captured notes descriptive titles
   - Batch-process: move into correct Project, Area, or Resource folders

5. **Choose tasks for the week**
   - Clear task manager inbox
   - Select specific tasks you're committing to this week

---

## Monthly Review Checklist

Perform to evaluate big-picture priorities and ensure alignment.

1. **Review and update goals**
   - Evaluate quarterly or annual goals
   - Cross off completed ones
   - Adjust scope where needed

2. **Review and update project list**
   - Archive completed projects
   - Add new ones
   - Ensure active list reflects current reality
   - Check the 10-15 project rule

3. **Review areas of responsibility**
   - Evaluate long-term areas (health, finances, relationships)
   - Are you meeting your standards?
   - Do any areas need new projects to address?

4. **Review someday/maybe tasks**
   - Check distant dreams and aspirations
   - Have changing circumstances made any actionable?
   - Promote to Projects if ready; remove if no longer relevant

5. **Reprioritize tasks**
   - Based on this holistic review, adjust day-to-day priorities

## Project List Audit

A guided exercise for when you feel overwhelmed, unfocused, or suspect your project
list has drifted. Run this quarterly or whenever things feel off.

### Step 1: List Everything

Write down every "project" you're currently tracking — formal or informal. Include
work projects, personal projects, side hustles, creative endeavors, home improvement,
learning goals, everything.

### Step 2: Apply the Project Test

For each item, ask two questions:
1. **Does it have a clear outcome?** (What does "done" look like?)
2. **Does it have a deadline?** (When does it need to be done by?)

If YES to both → it's a real Project. Keep it.
If missing a clear outcome → it might be an **Area** (ongoing responsibility) or a
**vague aspiration**. Demote it.
If missing a deadline → add one, or demote to Area/Resource.

### Step 3: Identify False Projects

Common false projects:
- "Get healthy" → This is an Area (Health). A project would be: "Complete 12-week
  strength program by June 15"
- "Learn Spanish" → This is a Resource interest. A project would be: "Complete
  Duolingo Spanish tree by March 30"
- "Organize photos" → Without a deadline, this languishes forever. Either set a
  deadline or admit it's a someday/maybe item.

Move false projects to Areas, Resources, or a dedicated "Someday/Maybe" list.

### Step 4: Identify Mega-Projects

Mega-projects are items so large they should be broken into phases:
- "Renovate house" → Phase 1: Kitchen (by April), Phase 2: Bathroom (by July)
- "Launch business" → Phase 1: Business plan (by March), Phase 2: MVP (by May)
- "Write book" → Phase 1: Outline (by April), Phase 2: First draft (by August)

Break mega-projects into individual project folders with their own deadlines.

### Step 5: Check the 10-15 Rule

Count your real projects. Then:
- **More than 15:** Your attention is too fragmented. Identify 3-5 projects to
  pause (write a Hemingway Bridge for each) or archive.
- **Fewer than 10:** You have room for more. Check Areas — is any standard slipping
  that needs a project? Check Someday/Maybe — is anything ready to activate?
- **10-15:** You're in the sweet spot.

### Step 6: Prioritize

From your refined list, identify:
- **Top 3 projects** — These get your best energy and most time this month
- **Active but lower priority** — These you advance when the top 3 are blocked
- **On deck** — Ready to start when a slot opens up

### Output

You should now have:
- A clean project list of 10-15 items
- Each with a clear outcome and deadline
- False projects demoted
- Mega-projects broken into phases
- A top 3 priority ranking

## Monthly Tidy-Up (15-Minute Routine)

A lightweight version of the monthly review:
- Close dormant projects (no activity in 30+ days — archive or revive)
- Promote repeated tasks to Areas if they represent ongoing responsibilities
- Prune stale Resources you're no longer interested in
- Archive old meeting notes that served their purpose

## Digital Detox / Ultimate Reset

When feeling overwhelmed by digital clutter:

### Step 1: Archive Everything
- Desktop, Downloads, Documents, cloud drives, notes inbox — move ALL into a single folder titled "Archive [today's date]"
- Delete nothing; just clear the decks

### Step 2: Clear Inboxes
- Archive all emails (or everything older than a week/month)
- Mark all messaging and social media DMs as read

### Step 3: Create a Project List
- Write down 10-15 active projects you are currently committed to
- This provides instant clarity on true priorities

### Step 4: Turn Off Notifications
- Shut off non-essential app notifications
- Set designated times to check devices
- Set intentional boundaries around screen time
