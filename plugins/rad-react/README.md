# rad-react

Modern React best practices for Claude Code. Functional components only — no class components.

Covers hooks, JSX, props, state, routing, forms, data fetching, server/client boundaries, accessibility, TypeScript, testing, performance optimization, security, and code organization.

## Skills

| Skill | Purpose |
|-------|---------|
| `react-foundations` | Components, hooks, JSX, props, state management, context |
| `react-app-building` | Routing, layouts, forms, data fetching, server/client boundaries |
| `react-engineering` | Project tooling, testing, code organization, TypeScript integration |
| `react-performance` | Re-render optimization, memoization, lazy loading, profiling |
| `react-accessibility` | ARIA patterns, keyboard navigation, screen reader support in React |
| `react-security` | XSS prevention, IDOR checks, input sanitization, secure data handling |

## Agents

| Agent | Purpose |
|-------|---------|
| `react-reviewer` | Reviews React code for anti-patterns, accessibility, performance, and security issues |
