# rad-brainstormer

Collaborative ideation using proven methodologies for Claude Code. Any topic: software, business, product, content, research.

Supports structured divergent/convergent phases, creative unblocking, domain research, idea evaluation, and design sprints.

## Skills

| Skill | Purpose |
|-------|---------|
| `brainstorm-session` | Facilitated brainstorming with structured divergent/convergent phases |
| `idea-generation` | Generate diverse ideas using multiple creative techniques |
| `idea-evaluation` | Evaluate and prioritize ideas using structured criteria |
| `creative-unblock` | Break through creative blocks with lateral thinking techniques |
| `scamper` | SCAMPER methodology — Substitute, Combine, Adapt, Modify, Put to other uses, Eliminate, Reverse |
| `six-hats` | Six Thinking Hats — explore all perspectives systematically |
| `reverse-brainstorm` | Reverse brainstorming — make the problem worse to find solutions |
| `five-whys` | 5 Whys root cause analysis |
| `how-might-we` | HMW reframing — turn problems into opportunity statements |
| `design-sprint` | Design sprint — structured spec creation and architecture design |

## Agents

| Agent | Purpose |
|-------|---------|
| `domain-researcher` | Research any topic domain via web search — landscape, approaches, constraints |
| `idea-challenger` | Stress-test ideas — pre-mortem analysis, feasibility/desirability/viability |
| `spec-reviewer` | Review design specs for completeness, consistency, and implementation readiness |
