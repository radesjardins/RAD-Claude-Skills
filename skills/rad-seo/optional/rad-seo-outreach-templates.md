# RAD SEO — Outreach Email Templates

Optional reference for link building outreach. Upload this alongside `rad-seo-strategy.md` if you're doing link building campaigns.

Customize every template with specific details about the recipient's site. Never send generic emails.

---

## Template 1: Broken Link Notification

```
Subject: Quick heads-up — broken link on [Page Title]

Hi [Name],

I was reading your article on [Topic] and noticed that the link to
[Dead Resource] in the [section name] section appears to be broken.

I recently published a guide on [Similar Topic] that covers [brief
description]. It might be a good replacement:

[Your URL]

Either way, wanted to flag the broken link. Great article!

[Your Name]
```

## Template 2: Skyscraper Outreach

```
Subject: [Topic] — updated resource you might like

Hi [Name],

I saw you linked to [Original Article] in your post about [Topic].
That piece is solid, but it was published in [Year] and some of the
information is outdated.

I just published an updated version that includes [new thing 1],
[new thing 2], and [new thing 3]:

[Your URL]

Thought it might be worth a look if you ever update that post.

Cheers,
[Your Name]
```

## Template 3: Resource Page Pitch

```
Subject: Suggestion for your [Topic] resources page

Hi [Name],

I came across your [Topic] resources page and found it really useful —
especially the section on [specific section].

I recently published [Content Title], which covers [brief value
description]. I think it could be a helpful addition for readers.

Here is the link: [Your URL]

Thanks for curating such a great list!

[Your Name]
```

## Template 4: Guest Post Pitch

```
Subject: Guest post idea — [Proposed Title]

Hi [Name],

I have been following [Site Name] and especially enjoyed the recent
piece on [Recent Article]. It got me thinking about a related angle
readers might find valuable.

Proposed topic: [Title]

What it would cover:
- [Point 1]
- [Point 2]
- [Point 3]

About me: [One sentence credential]. See my writing at [link].

Would this be a good fit? Happy to adjust the angle.

[Your Name]
```

## Template 5: Link Reclamation Request

```
Subject: Thanks for mentioning [Brand Name]!

Hi [Name],

I noticed you mentioned [Brand Name] in the article about [Topic] —
thank you for the shoutout!

Would you mind adding a link to the site so readers can find it easily?
The homepage is [URL], or link directly to [relevant page]
which is most relevant to what was mentioned.

Really appreciate it!

[Your Name]
```

## Template 6: Roundup Submission

```
Subject: Submission for your [Topic] roundup

Hi [Name],

I have been reading the [weekly/monthly] roundup and always find great
stuff through it. I would love to submit a piece for consideration:

Title: [Content Title]
URL: [Your URL]
Why it fits: [One sentence explaining the value for the audience]

Thanks for putting these together!

[Your Name]
```

## Outreach Best Practices

1. **Personalize every email.** Reference something specific about their site or recent article.
2. **Keep it short.** Under 150 words. Respect their time.
3. **Lead with value.** Offer something useful (broken link fix, better resource) before asking.
4. **One follow-up only.** Send 5-7 days after the first email. More than that is spam.
5. **Track everything.** Log date, recipient, tactic, status, and result in a spreadsheet.
6. **Send from a real person.** Use your name, not "marketing team" or a no-reply address.
7. **Test subject lines.** Keep them short, specific, and curiosity-driven.

| Tactic | Time Investment | Success Rate | Best For |
|--------|---------------|-------------|----------|
| Broken Link Building | 8-12 hrs/campaign | 5-10% | Established niches |
| Skyscraper Technique | 15-20 hrs/campaign | 5-15% | Competitive niches |
| Resource Page Outreach | 5-8 hrs/campaign | 5-15% | Educational, B2B |
| Guest Posting | 8-15 hrs/post | 10-30% acceptance | Strong writers |
| Link Reclamation | 2-4 hrs/batch | 20-40% | Established brands |
| Roundup Submissions | 2-3 hrs/week | 10-25% | Active blogging niches |
| HARO / Media Requests | 3-5 hrs/week | 5-20% per pitch | Subject experts |
| Testimonial Building | 1-2 hrs each | 30-50% | Quick wins |
| Podcast Guesting | 3-5 hrs each | 20-50% booking | Comfortable speakers |
| Digital PR | 20-40 hrs/campaign | DA 50-90+ | Brands with unique data |
